## react-boilerplate

### 框架功能
+ rem 布局
+ 使用 Airbnb 代码规范做检查 
+ 三方 UI 组件支持 rem 布局无需重置样式
+ 支持 css-modules
+ .styl 下样式直接转换为 rem 
+ 路由代码分割
+ 不可变数据结构
